(1)
# Check the version of node on your local installation
node -v

(2)
# Create a template react app
npm create vite@latest my-react-app -- --template react

(3)
# Install in the my-react-app directory 
npm install gh-pages --save-dev

(4)
Open the package.json file and add a homepage property and new scripts entries.
Add a homepage URL at the top level of the JSON object, formatted as 
https://<username>.github.io/<repository-name>/.

"homepage": "https://{username}.github.io/{repo-name}",

"homepage": "https://benbraniff.github.io/JackiesGarden",

(5)
Add Deployment Scripts: In the same package.json file, find the "scripts" section and add these two lines:

JSON
"scripts": {
  "predeploy": "npm run build",
  "deploy": "gh-pages -d build",
  ...
}

(6)
npm run deploy


(7)
Make sure the repo is Public
Caution: This repository is private but the published site will be public.

(8)
# In the github repo > settings > Pages
Deploy from a Branch >
gh-pages /(root)

(9)
# in the "vite.config.js" file add the base

export default defineConfig({
  plugins: [react()],
  base: "/JackiesGarden/"
})
